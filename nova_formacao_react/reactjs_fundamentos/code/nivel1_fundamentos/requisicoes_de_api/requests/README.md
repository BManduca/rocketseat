# Requisições de API

- Inicialmente esse vai ser um projeto com foco na prática.
- Um dos primeiros passos foi configurar um JSON Serverm para simular requisições HTTP com um banco de dados em JSON.
- Na raiz do projeto foi criado um arquivo db.json para armazenar usuários e configurar endpoints automáticos pra GET e POST, facilitando assim o desenvolvimento, especialmente quando a API não estiver pronta.
